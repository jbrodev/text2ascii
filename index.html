Convert any text into large ASCII art banners or chunky Unicode block letters.

Paste the result into your terminal, README, or anywhere that uses a monospace font.

NOTES
  • Copy the result and paste it into your terminal, shell script, or README.
  • For terminal use, make sure your font supports Unicode (try Consolas, Fira Code, or Cascadia Code).
  • ASCII art mode requires an internet connection on first use to load fonts from CDN.
  • Unicode block mode works fully offline.
  • Install the CLI tool: pip install git+https://github.com/jbrodev/text2ascii.git
